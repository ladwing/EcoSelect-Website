# 🌿 EcoSelect Website

This repository contains the frontend landing page and privacy policy for **EcoSelect**, an AI-powered Chrome extension that helps users find sustainable shopping alternatives.

## 🔗 Live Site
[https://ecoselect.shop](https://ecoselect.shop)

## 📁 Project Structure
* `index.html` - The main landing page outlining features, how it works, and download links.
* `privacy.html` - The privacy policy required for Chrome Web Store compliance.
* `logo.png` - The EcoSelect brand asset.

## 🛠️ Tech Stack
* HTML5
* Vanilla CSS
* Hosted via Vercel
